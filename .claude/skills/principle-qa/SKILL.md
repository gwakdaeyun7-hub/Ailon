# Principle QA -- 학문스낵 curated 콘텐츠 품질 검토 및 수정

학문스낵 curated 원리 콘텐츠(`scripts/curated_principles/*.md`)의 품질을 Gold Standard인 **Simulated Annealing**(opt_simulated_annealing.md)과 비교하여 검토하고, 기준 미달 콘텐츠를 수정합니다.

## 사용법

- `/principle-qa` -- 미검토 또는 이전 B 이하 등급 자동 선택
- `/principle-qa opt_simulated_annealing` -- 특정 원리 검토
- `/principle-qa all` -- 전체 45개 일괄 검토 (리포트만, 수정은 B 이하만)
- `/principle-qa fix neuro_attention_mechanism` -- 검토 없이 바로 수정

## Gold Standard 기준 (SA 참조)

**참조 파일:** `scripts/curated_principles/opt_simulated_annealing.md` (4,542자 KO, ~9분 읽기)

### 구조 필수 요소

| 요소 | 기준 |
|------|------|
| Front-matter | difficulty, connectionType, keywords, keywords_en |
| 제목 줄 | "English Name - 한줄 정의" (콘텐츠 첫 줄) |
| 학문 원리 섹션 | 원래 학문의 원리를 AI 언급 없이 독립적으로 설명 |
| 알고리즘 번역 섹션 | 학문 원리 -> AI 알고리즘 대응 관계 명시 |
| 핵심 메커니즘 | 수학적/논리적 핵심 (수식 포함 가능) |
| 현대 AI 연결 | 현대 AI 기법에 어떻게 살아있는지 |
| 한계/약점 | 솔직한 한계 평가 3-4개 |
| 용어 정리 | 전문 용어 5-10개 (term - definition 형식) |
| ---EN--- 섹션 | 번역체 아닌 자연스러운 영문 버전 |

### 길이 기준

- KO 본문: 4,242~4,842자 (SA +-300자)
- EN 본문: KO의 70~130%
- 섹션 수: 6-9개 (## 기준, 최소 5개)
- 용어 정리: 5-10개 (최소 4개)

### 콘텐츠 깊이 등급 기준 (7개 차원)

| 차원 | A (SA 수준) | C (미달) |
|------|------------|---------|
| 학문 원리 설명 | 원래 학문 맥락에서 독립 이해 가능, 물리/생물/수학적 직관 명확 | AI와 바로 연결하여 원래 학문 깊이 부족 |
| 역사적 맥락 | 핵심 인물+연도+원래 목적+후속 발전 | 역사적 맥락 거의 없음 |
| 수식/수학적 표현 | 핵심 수식+각 변수 설명+물리적 의미 | 수식 없이 말로만 설명 |
| AI 연결 구체성 | 항목별 대응+직접 영감 vs 구조적 유사성 구분 | "AI에 활용된다" 수준 추상적 |
| 공간적 비유 | 복잡한 개념을 공간적 비유로 시각화 | 비유 없이 추상적 설명 |
| 한계 평가 | 구체적 한계 3-4개+왜 한계인지 | 한계 섹션 없음 |
| 연결의 정직성 | 직접 영감/구조적 유사/독립 발전 정확히 구분 | 과장된 인과 관계 |

### 명확성 기준

- 전문 용어는 처음 등장 시 풀어서 설명
- 복합 개념은 2-3문장으로 분리
- 추상적 원리마다 구체적 사례 1개+
- 인과 계보 과장 금지 (독립 발전은 독립 발전으로)
- 수식은 최종 수식만이 아닌 각 항의 의미 설명
- 대상 독자: AI 관심 일반인~초중급

### SnapsContentRenderer 호환성

| 지원 | 미지원 |
|------|--------|
| `## 제목`, `**굵은 글씨**`, `- 불릿`, `1. 번호`, 코드/수식 블록, `용어 - 정의`, 유니코드 수학 기호 | 취소선, 이탤릭, 링크, 테이블, 이미지, HTML, LaTeX, 중첩 리스트 |

수식 감지: 80자 이하, 한글 25% 미만, `=`/`e^`/`P(...)`/그리스 문자 등 수학 패턴 포함

## 검토 프로세스 (8개 평가 차원)

### 1. 구조 검사 (자동)
Front-matter, 제목 형식, 섹션 수, ---EN--- 존재, 길이, EN 비율, 용어 수, 렌더러 호환성 확인.

### 2. 학문적 정확성 (ai-interdisciplinary-scholar 에이전트 필수 사용)
원리 설명 정확성, AI 매핑 정확성 (직접 영감 vs 구조적 유사 구분), 누락 핵심 개념, 잘못된 연결.

### 3. 콘텐츠 깊이
위 7개 차원에 A/B/C 등급. SA와 비교하여 판정 근거 제시.

### 4. 명확성
용어 선설명 누락, 복합 문장, 사례 부족, 인과 과장, 수식 중간 단계, 난이도 적절성.

### 5. 길이 및 밸런스
KO 4,242~4,842자, 원리 설명 30-40%, AI 연결 25-35%, 한계+용어 15-20%, 원리>=AI (50:50~60:40).

### 6. 원리 간 중복
같은 super_category/discipline 내 AI 연결 중복, 설명 패턴 중복, 용어 정의 일관성.

### 7. 바이링구얼 품질
EN 자연스러움 (번역체/수동태 과다 지적), KO/EN 내용 일치, 고유명사 일관성 (음차 금지), EN 용어 정리.

### 8. 렌더러 호환성 (자동)
미지원 마크다운, 80자+ 수식, 한글 25%+ 수식, definition 오인식.

## 수정 (B 이하만)

- **학문적 내용 수정**: ai-interdisciplinary-scholar 에이전트 사용 (KO+EN 동시 수정)
- **구조/편집**: 직접 수행
- **수정 원칙**: SA 참조하되 복사 금지, A등급 차원 보존, 최소 범위 수정, 렌더러 호환 유지, 길이 범위 준수, EN 동기화
- **우선순위**: 학문적 정확성 > 콘텐츠 깊이 > 명확성 > 기타
- **SA(opt_simulated_annealing.md)는 절대 수정 금지** -- Gold Standard 참조 전용

## 주의사항

- 에이전트 병렬 실행 가능 (여러 원리 동시 검토)
- 일괄 시 super_category별로 분할 실행 (공학->자연과학->형식과학->응용과학)
- 수정은 기존 파일 Edit, 새 파일 생성 금지
- 수정 후 `_load_curated_content()` 호환성 + principle_seeds.py 교차 확인
- 종합 A 등급은 리포트만 제공, 수정하지 않음
