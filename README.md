# Ailon

A personal project built with Next.js and Firebase.

## Tech Stack

- Next.js 14 (App Router) + TypeScript + Tailwind CSS
- Firebase (Auth, Firestore)
- Python (automation scripts)
- Vercel (hosting)

## Setup

```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
```
